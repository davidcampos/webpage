Tools

Egas

Collaborative and interactive curation of biomedical documents.

Egas is a web-based platform for biomedical text mining and collaborative curation, supporting manual and automatic annotation of concepts and relations.

Becas

Biomedical concept recognition web-services and visualization.

Becas is a web-based tool, web-services and widget for biomedical concept identification, supporting the annotation of MEDLINE abstracts and free text documents with eleven biomedical concept types.

Neji

Framework and tool for heterogeneous biomedical concept recognition.

Neji is an open source, flexible, easy and powerfull framework and tool for faster biomedical concept recognition. It allows automatic extraction of dozens of heterogeneous biomedical concepts using the most appropriate and optimized techniques. When evaluated on available corpora, Neji outperformed existing solutions.

Gimli

Machine learning-based biomedical named entity recognition.

Gimli is an open source off-the-shelf and ready to use tool and library for complex named-entity recognition, providing trained and optimized models for recognition of biomedical entities from scientific texts. When evaluated on available corpora, Gimli outperformed existing open source solutions.

Totum

Machine learning-based harmonization of biomedical concept annotations.

Totum is a machine learning-based solution to harmonize annotations of biomedical concepts provided by heterogeneous systems. It was optimized and evaluated against a combination of manually curated corpora, presenting significant improvements when compared with state-of-the-art approaches.

Trigner

Machine learning-based event trigger recognition.

Trigner is an open source solution for biomedical event trigger recognition, taking advantage of a completely configurable algorithm to automatically optimize the feature set and training parameters for each event type. When evaluated on available corpora, Trigner sifnificantly outperformed available solutions on simple events.
